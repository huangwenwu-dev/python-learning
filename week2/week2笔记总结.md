# 第二周笔记：函数 + 面向对象

## 一、用法速查（我要吃透的）
- 函数：`def` 定义/调用、形参 vs 实参、`return` 返回值
- 进阶：默认参数、关键字参数、返回多值（元组拆包）、作用域（局部 vs 全局）
- 类：`__init__`、`self`、实例方法
- 继承：`super().__init__()`
- 封装：`_` 单下划线 / `__` 双下划线 + getter/setter
- 综合：Student 成绩管理类（平均分 / 最高分）

## 二、关键直觉（一句话记牢）
- 函数像奶茶店配方：给原料（参数），按步骤做，`return` 把成品端出来
- `print` 只是"贴在墙上给人看"，`return` 才是"把成品交回给程序继续用"
- `self` 就是"这一杯自己的料"：存实例属性必须 `self.xxx`
- 子类写了自己的 `__init__`，就得先 `super().__init__()` 把父类的料备齐

## 三、本周踩坑记录
- 把 `print` 当 `return` 用（函数算完没 return，外面拿到 `None`）
- 存属性忘加 `self.`
- 子类定义了 `__init__` 却忘调 `super().__init__()`

## 四、回头补的盲区
- 类方法 / 静态方法、`@property` 装饰器 → 用到再深入（目前 getter/setter 够用）
