## 🤖 我的 AI 使用守则（第 3~10 周）
我用 AI 是为了"学得更快"，而不是"逃避思考"。所以我约定：
1. **AI 只当"答疑老师"，不当"代码枪手"。**
   可以问概念、问思路、让它帮我看报错，但不让它直接替我写完整功能。
2. **先自己想、自己写，卡住了再问。**
   每个问题至少独立尝试 15 分钟，再求助 AI。
3. **抄来的每一行都要看懂。**
   凡是参考 AI 的代码，我必须能逐行解释、能自己默写出来——否则就是没学会。
4. **报错先自己读。**
   先看错误类型和行号、自己搜，再问 AI"为什么会这样"。
5. **定期"关掉 AI"自测。**
   每周至少独立完成一个小程序，全程不看任何提示。
